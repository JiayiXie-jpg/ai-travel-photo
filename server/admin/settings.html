系统设置

配置 AI 生成时使用的 Prompt。每个模板可单独配置 Prompt（在模板管理页编辑），未配置的模板将使用以下全局默认值。

AI旅拍 - 全图换脸 Prompt

prompt_travel_full

当旅拍模式使用全图换脸策略时（人脸检测失败的 fallback），使用此 Prompt。Image 1 = 用户照片，Image 2 = 模板照片。

AI旅拍 - 裁剪换脸 Prompt

prompt_travel_face

当旅拍模式使用裁剪-换脸-贴回策略时（人脸检测成功），仅对裁剪出的脸部区域使用此 Prompt。

AI试衣 Prompt 模板

prompt_tryon

AI试衣模式使用此 Prompt。支持以下占位符，会根据用户选择自动替换：

${personDesc} - 人物描述，如 "the person in image 1 (a young adult aged 19-30, with a slim and slender body type)"
${bodyDesc} - 体型描述，如 "slim and slender"、"slightly chubby"、"plus-size"，未选择时为 "natural"
